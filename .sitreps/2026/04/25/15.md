## Lessons

- **Cross-branch contamination from shared main tree**: test-agent committed
  Slice C onto stanza's #165 branch because both were operating in the main
  tree. Worktree-per-feature isolation prevents this; #165 is now the
  load-bearing slice for the topology.
- **Agents can't `cd` themselves into worktrees mid-session**: cwd is fixed
  at session launch. Real fix is `c2c start --worktree` exec'ing with the
  right cwd. Logged into #165 scope.
- **Restart bug class**: respawning a long-running outer process in the
  caller's context creates pane drift. Should not be done — signal the
  inner PID and let the existing supervisor handle relaunch.
