## What This Project Is

`c2c-msg` is a small local toolkit for discovering Claude sessions, injecting messages into them, and reading recent transcript history.

## What This Project Is Not

- not a built-in Claude feature
- not a Teams-based relay
- not yet a fully robust autonomous multi-agent runtime
