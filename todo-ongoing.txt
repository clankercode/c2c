# Ongoing TODOs

This doc keeps track of long term ongoing tasks and projects. It should be updated as progress is made. Each long term project should have an folder under `.projects/` where we keep track of logging the specifics of what we've done, decisions made, tasks remaining, etc. Each section here in this document should be a short summary of recent progress and the current state of the task/project. Next steps can be documented too, though the full future steps / plans / etc go in the project folder. Each summary should also include a short goal statement (though longer and more detailed goals go in the project folder).

## Maintenance

- **Edit cadence**: touched by whichever agent ships a change that affects a project's state. coordinator1 sweeps during sitreps to ensure summaries reflect reality.
- **Statuses**: `active` (work in flight) · `blocked` (waiting on external / human) · `paused` (deprioritized) · `done`.
- **Entry shape**: goal, status, folder link (or `**todo**` if the folder doesn't exist yet), current state in ≤3 sentences, next step.
