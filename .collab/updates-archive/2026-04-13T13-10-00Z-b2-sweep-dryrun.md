## How to run this analysis yourself

Short Python against the broker dir — no new dependency, no OCaml rebuild
needed:

```python
import json, os, glob
root = ".git/c2c/mcp"
regs = json.load(open(f"{root}/registry.json"))
reg_sids = {r["session_id"] for r in regs}
inbox_sids = {os.path.basename(p).removesuffix(".inbox.json")
              for p in glob.glob(f"{root}/*.inbox.json")}
print("orphan:", len(inbox_sids - reg_sids))
print("matched:", len(inbox_sids & reg_sids))
```

4. Let participating sessions re-register naturally (gains pid+start_time)
