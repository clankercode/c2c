**Date**: 2026-04-23T19:58:00Z

**Type**: classification audit

## Scope

All Tier 3 commands in `command_tier_map` evaluated for correct classification.

## Classification Framework

- **Tier 1**: Safe for agents (messaging, queries, no side effects)
- **Tier 2**: Safe with care (lifecycle, side effects, setup)
- **Tier 3**: Unsafe for agents (systemic impact, requires operator)
- **Tier 4**: Internal plumbing (never shown in agent help)

## Commands Evaluated

### KEEP Tier 3

| Command | Reason |
|---------|--------|
| `restart-self` | Kills supervisor process — catastrophic if run inside managed session |
| `relay-serve` | Runs relay server — operator infrastructure |
| `relay-gc` | Relay garbage collection — operator infrastructure |
| `relay-connect` | Starts connector daemon as child process — operator setup |
| `setcap` | Grants CAP_SYS_PTRACE, requires sudo — dangerous |
| `gui` | Launches Tauri GUI — agents use CLI/MCP |
| `diag` | Operator diagnostics — not agent-facing |
| `smoke-test` | Runs integration tests — operator verification |

| `relay-setup` | Tier 3 | Tier 2 | Agent configures relay connection |

## Key Design Notes

1. **Tier filter only applies to top-level commands.** Subcommands inherit group visibility. Once `relay` is Tier 2, all subcommands are accessible — the per-subcommand tier is documentation/operator-enforcement, not enforced at the CLI layer.

2. **`install` already fixed** (`a107a6f`): graduated to Tier 2 in prior session. Confirmed working — agents can now self-configure.

3. **Tier 3 relay ops remain dangerous**: `relay serve`, `relay gc`, `relay connect` are still in the group but require operator intent to invoke.

## Principles for Future Command Classification

When adding new commands, ask:
1. **Could an agent session legitimately need this?** (Tier 1 or 2)
   - Messaging, queries, registration → Tier 1
   - Setup, configuration, lifecycle → Tier 2
2. **Does this have systemic impact or require operator context?** → Tier 3
   - Daemon spawning, sudo operations, infrastructure → Tier 3
3. **Is this internal plumbing called by plugins?** → Tier 4
